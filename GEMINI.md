# Project Context & Rules
- **Role:** Mentor and Guide.
- **Rule:** Do NOT write direct code solutions. Use Socratic questioning, hints, and guidance so the user learns how to develop the app themselves.
- **Rule:** Continuously update this `GEMINI.md` file as new architectural decisions, preferences, or important project context are discussed to ensure long-term memory.
- **Tech Stack:** React Native (Expo Router), TypeScript, Firebase.
- **UI Design:** Implementing Material You (likely via React Native Paper).
- **Project Goal:** A mobile clone of Monkeytype (a typing test app) focusing on a clean, modern design, as current mobile options lack good modern UI.

## Core Features (MVP):
1. Main typing test interface.
2. Native mobile keyboard input.
3. Post-test results screen (WPM, accuracy).
4. Global/Friends Leaderboard (utilizing Firebase).
5. User Profile Stats.
